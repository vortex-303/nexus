Features

Everything your team needs.

Chat, tasks, docs, AI agents, and a knowledge base that builds itself — all in one place.

Threaded chat

Channels, DMs, reply threads, emoji reactions, @mentions with autocomplete, desktop notifications, drag & drop file sharing, typing indicators, and markdown with syntax highlighting.

Tasks

Create tasks from conversations with a click. Assign, set deadlines, track progress. Brain automatically follows up when deadlines approach.

Documents

Rich text editor with a built-in knowledge base. Import from URLs. Brain reads every doc, creates new ones, and links related knowledge automatically.

AI Agents

8 built-in templates: writer, coder, analyst, planner, researcher, and more. Define your own in markdown — give each agent skills, tools, and a persona.

MCP Tools

Extend agents with Model Context Protocol servers. Web search, calendars, databases, APIs — if there's an MCP server for it, Nexus can use it.

9 roles from guest to super admin, 22 granular permissions, and an org chart. Control exactly who sees what, who can invite, and who manages agents.

Open Nexus Cloud ~10 seconds

Install Script

One-line installer for Linux & macOS. Downloads the latest release binary.

$ curl -fsSL https://raw.githubusercontent.com/vortex-303/nexus/main/install.sh | sh
$ nexus serve
~30 seconds

Docker

Single container, persistent volume. Works on any Docker host.

$ docker run -d -v nexus-data:/data \
  -p 8080:8080 ghcr.io/vortex-303/nexus
~1 minute

Build from Source

Clone the repo, build, run. Requires Go 1.25+ and Node 22+.

$ git clone https://github.com/vortex-303/nexus
$ cd nexus && make build && ./nexus serve
~2 minutes
